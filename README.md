# Job Portal Fullstack App
A fullstack Job Portal application built using React, Node.js, Express, and MongoDB Atlas. This project allows users to create, view, update, and delete job listings.

## Tech Stack 
- Frontend: React (vite)
- Backend: Node.js, Express
- Database: MongoDB Atlas(cloud)
- Styling: CSS

## Features
- Add new job postings
- View all jobs
- Update job details
- Connected to cloud database (MongoDB Atlas)

## Project Structure
job-portal/frontend/backend/
---
## Setup Instructions
### 1. Clone the repository
git clone


### 2. Install dependencies
Frontend:
cd frontend npm install npm run dev
backend:
cd backend npm install node index.js
---

## Environment Variables
Create a `.env` file in backend and add:
MONGO_URI=your_mongodb_connection_string
